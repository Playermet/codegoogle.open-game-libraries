## Features ##
  * Surround sound via OpenAL Soft
  * Device enumeration
  * Ogg/Vorbis & Wave file loading ( streamed )
  * Multichannel audio emitters
  * Directional speakers
  * Doppler effects
  * Soon environmental effects
